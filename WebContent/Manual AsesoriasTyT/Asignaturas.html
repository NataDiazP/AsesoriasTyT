Gestión de Asignaturas

Esta opción permite administrar las asignaturas.

Campos
 
Código Asignatura

Corresponde al código que identifica la asignatura.
Nombre Asignatura

Corresponde al nombre de la asignatura.

   
Botones
 

Permite almacenar una nueva aula.

Permite consultar un aula por medio del código y bloque
 

Permite modificar el nombre de un aula
 

Permite eliminar un aula existente
